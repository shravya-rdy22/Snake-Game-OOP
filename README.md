 Snake Game (Java OOP + GUI)

Overview
This project is a real-time Snake Game developed using Java and Swing. It demonstrates object-oriented programming concepts, event-driven architecture, and GUI-based application development.

Features
- Real-time snake movement using Timer
- Keyboard controls (Arrow keys)
- Dynamic snake growth on eating food
- Collision detection (wall & self)
- Score tracking system
- GUI-based gameplay using Java Swing
- Random food generation (avoiding snake body)

 Technologies Used
- Java (Core Programming)
- Java Swing (GUI Development)
- AWT (Graphics & Event Handling)
- OOP Concepts (Encapsulation, Inheritance, Polymorphism)
- Data Structures (LinkedList)

 Concepts Implemented
- Event-driven programming
- Game loop using Swing Timer
- Collision detection logic
- Real-time rendering using Graphics
- Keyboard input handling (KeyListener)

 Controls
Key : Action 
⬆️ : Move Up 
⬇️ : Move Down 
⬅️ : Move Left 
➡️ : Move Right
